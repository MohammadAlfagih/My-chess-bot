# My Chess Bot ♟️🤖

A machine learning project to build a chess engine that plays like me! 
This bot is trained on my personal games from Chess.com.

## Phase 1: Data Collection
The `Data-Collection.py` script automatically fetches all archived games for multiple accounts via the Chess.com public API and saves them locally in organized JSON files.

## How to run
1. Clone the repository.
2. Install requirements: `pip install -r requirements.txt`
3. Run the script: `python Data-Collection.py`
